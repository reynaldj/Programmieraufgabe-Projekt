Diese Aufgabe geht darum, eine Oktadoku zu lösen. In der Dokumente in* kann man Beispiele für Oktadoku-Rätsel finden. 
Man kann auch die Oktadoku so lösen, sodass die Ziffer in der Diagonal auch unterschiedlich sind, in dem man das Program mit dem Argument -x laufen.
In der Dokumente output* und output*mitDiagonal kann man die entsprechende Lösung sehen.
